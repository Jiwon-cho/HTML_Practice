태그별 배치속성

태그별로 block속성과 inline 속성으로 구분할 수 있음. block: 한 라인 전체를 차지함-> 개행이 됨.width height 조정 가능 inline: 라인에서 여러 태그를 표현 가능 -> 개행 되지 않음 width height조정 불가능 크기는 라인에서 지정하는대로 처리함. 기본태그의 배치속성을 변경할 수 있음 -> display 속성을 이용
