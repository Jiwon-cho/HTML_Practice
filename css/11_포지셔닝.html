position 속성

태그를 페이지상에서 원하는 위치에 배치할 수 잇게 하는 속성 position속성은 위치를 지정하는 속성인 top,left,right,bottom 과 같이 사용함. position 속성값 static: 기본값 position을 설정하지 않으면 기본 설정 relative: 부모의 position 속성이 static 일 때 부모태그의 왼쪽 꼭지점 지점이 기준(0,0) absolute: 부모의 position 속성이 static일 때 페이지의 왼쪽 꼭지점 지점이 기준(0,0) *부모의 position속성이 releative, absolute명 둘다 부모의 왼쪽 꼭지점이 기준(0,0)
