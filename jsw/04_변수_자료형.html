변수선언

변수를 선언하는 방법 var 예약어를 이용해서 선언을 함.
var 변수명; *자료형을 선언하지 않음!
그러면 자바스크립트에는 자료형이 없나? 아니다 잇다 변수를 선언할 때 사용하지 않을뿐이다.
변수의 자료형이 결정되는 순간은 변수에 값이 대입됐을때 대입되는 값의 자료형을 가지고 결정
var 변수명; * 자료형이 정해지지 않은 상태 undefined 변수명=10; -> 이순간 변수명 변수의 자료형이 number로 결정이됨.

변수의 자료형별 리터럴

  • 문자열(String): " " ||' '
  • 숫자(Number): 10||180.5
  • 배열(Array): []||new Array();
  • 객체(Object): {}||new Object();
  • undefined: undefined
  • 함수(function): function(){}||()=>{}

변수의 활용 범위

변수를 접근할 수 있는 범위에 따라 영역(scope)을 구분 global scope: 페이지내에서 접근이 가능한 변수 -> window 객체의 멤버변수로 된 변수 local scope: 특정 함수호출시에만 접근이 가능한 변수->함수 내부에서 선언된 변수 *{}에 따른 범위는 없다! -> es6 보완! const,let 추가

자바스크립트는 블록범위{내의 영역}를 허용 하지 않음

변수명을 중복을 선언할 수 있음

변수명을 페이지에서 중복해서 선언한다고 에러를 발생하지 않는다. 있는 변수에 덮어 씌움

es6에서 보완한 변수 선언 예약어

let : 변경이 가능한 변수를 선언할때 쓰는 예약어(변경되는 데이터) const : 변경이 불가능한 변수를 선언할때 쓰는 예약어(함수저장, 고정값) 1. 블록범위를 허용함 2. 중복명칭이 있는지 확인
