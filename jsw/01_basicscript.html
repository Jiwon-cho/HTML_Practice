자바스크립트 선언하기

자바스크립트는 script태그안에 작성 하여 이용을 함 script 태그는 선언된 위치에 상관없이 html 페이지 안에 있으면 됨. script 태그는 여러개 선언해도 상관이 없다. 단 위에서 부터 순차적으로 실행을 한다.

자바스크립트 이용하는 방식

자바스크립트를 이용하는 방식은 3가지 방식이 있음

  1. inline: 태그내부의이벤트 속성(on ***)에 직접 javascript 코드를 작성하는것 * 간단한 구문만 사용
  2. internal: html페이지 내부의 script 태그를 선언하고 script 태그안에 javascript 코드를 작성하는 것
  3. external: ooo.js 파일을 만들어 javascript 코드를 작성하고 html 페이지에서 script 태그 src 속성으로 그 파일을 불러와 적용하는 것 * 내부로컬 파일 / 외부 http 통신방식(url 주소)으로 불러올ㄹ 수 있음.
