태그에 접근하는 방법

document객체의 메소드를 이용해서 접근을 할 수 있다. js에서는 페이지자체를 하나의 객체로 보고 페이지내부의 태그드르도 모두 단일 객체로 본다(DOM)

  • id값을 기준으로 태그한개를 가져오는 방법: getElementById()
  • class값을 기준으로 다수태그를 가져오는 방법 : getElementsByClassName()
  • tagName값을 기준으로 다수태그를 가져오는 방법 : getElementsByTagName()
  • namer값을 기준으로 다수태그를 가져오는 방법 : getElementsByName()

다수태그를 가져오는 매소드는 배열(List)방식의 객체를 반환 한개태그를 가져오는 매소드는 태그(Element객체)를 반환함

id값을 기준으로 태그 가져오기

태그의 id속성의 값을 기준으로 한개의 태그만 가져옴 document.getElementById("id값"); -> html태그중 id값이 인수값과 같은 태그를 가져옴

css 선택자로 태그를 불러오는 방법

css에서 사용하는 선택자를 이용해서 태그를 지정하여 가져오는 것 #test -> id 속성이 test인 태그 .test -> class 속성이 test인 태그 [type=text] -> type 속성이 text인 태그 ul>li -> ul 에 직접 연결된 li 태그
document.querySelector("선택자") -> 한개의 태그만 가져오는 것, 다수가 있으면 제일 위에 선언된 태그만 가져옴.첫번째 태그
document.querySelectorAll("선택자") -> 한개 이상의 태그를 가져오는 것
